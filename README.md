# Nebula
> A multi platform Game Engine being made from scratch in C++

### How to Run (Windows/Linux)

#### Dependencies
- Python
- GCC or CLANG (with a toolchain like MSVC on Windows)

#### Compiling

1. Clone the repository:
```bash
git clone https://github.com/neverevis/Nebula
cd Nebula
```

2. Change the settings on build.py (Compiler, Debug, Dependencies)
> Planing to make a json configuration in the future

3. Compile and run:
```bash
python build.py
./sandbox
```

### This project has a .vscode folder configuration and can be opened on VSCode
Onde the Folder is opened inside VScode, it can be compiled with `Ctrl + Shift + B` and executed with `F5`.
